# Oshikatsu Manager

推し活のイベント参加履歴や支出を管理するためのWebアプリです。  
ライブ・イベント・チェキなどの「推し活」を記録し、推しごとの履歴や支出を管理できます。

個人開発のポートフォリオとして作成しました。

---

## アプリ概要

推し活をしていると

・どのライブに行ったか  
・チェキやグッズにいくら使ったか  
・推しごとの履歴

などが分からなくなりがちです。

このアプリでは推しごとの履歴と支出を一元管理できます。

---

## 主な機能

- 推し登録
- ライブ / イベント履歴の登録
- チェキ・グッズなどの支出管理
- 推し活履歴の一覧表示
- 支出データの管理
